Core Decision

The Right Model for You

Not CaratLane's model. Your model — built lean, fast, and designed to scale without pain.

Recommended Architecture
Serverless-first, monolith now, split later
Start with a single Next.js app — frontend + API routes in one deployment. Zero backend server cost. Zero DevOps overhead. When you hit 10K/day, you pull out the heavy services (search, payments, notifications) into separate workers. No rewrite needed — just extract. This is how Vercel, Linear, and most modern startups at your stage operate. You save ₹8,000/month compared to running separate backend servers from day one.

Technology Choices

Every Layer Explained

Each choice is the best option for low cost + high performance + easy scale in 2026.

Category Technology Free Tier Why This, Not Alternatives
Framework Next.js 15 ✓ Free on Vercel SSG + ISR + SSR in one. Best SEO. API routes replace separate backend at 1K/day. You already know it.
Language TypeScript 5 ✓ Free Shared types between frontend and API. Catches bugs before deploy. Standard in 2026.
Styling Tailwind CSS v4 ✓ Free v4 is 10x faster build. Tiny CSS output. Design system tokens built in. No runtime cost.
Database Supabase PostgreSQL ✓ 500MB free vs Neon: Supabase includes Auth, Storage, Realtime in one bill. Simpler for your scale. Auto-backups.
ORM Drizzle ORM ✓ Free Fastest Node.js ORM. Type-safe SQL. Works perfectly with Supabase and Neon. Tiny bundle.
Auth Better Auth ✓ Free (self-hosted) vs NextAuth: supports OTP/phone login natively. RBAC built in. Works with Drizzle. Perfect for your anon-checkout flow.
Cache Upstash Redis ✓ 10K req/day free Serverless Redis — pay per request. No always-on server. Product cache, cart, rate limiting, sessions. ₹0 at launch.
Search Typesense ✓ Self-host free vs Algolia: free self-hosted option on Railway (₹500/mo). <50ms results. Filters, facets, typo-tolerant. Sync from Supabase via webhook.
Images Cloudinary ✓ 25GB free Auto WebP/AVIF, resize on-the-fly, face crop for fashion. CDN included. 25GB free covers 1000s of product images.
Storage Supabase Storage ✓ 1GB free Included in Supabase. Row-level security on files. CDN-backed. Enough for receipts, CSVs, admin uploads.
Payments Razorpay ✓ No setup fee 2% per transaction. UPI, cards, netbanking, COD, EMI. Best for India. Webhook-based — reliable.
Email Resend ✓ 3,000/mo free 3K emails/month free. Developer-first API. React email templates. Perfect for order confirmations.
SMS / OTP MSG91 ₹0.16/SMS Cheapest reliable OTP for India. DLT registered. ₹160 for 1,000 OTPs. No monthly fee.
CDN / Security Cloudflare Free ✓ Free forever DDoS protection, WAF, global CDN, SSL — all free. Handles traffic spikes without your servers breaking.
Hosting Vercel Pro ₹1,700/mo Auto-scales, zero DevOps. Preview deploys. Edge functions. Worth ₹1,700 vs managing your own server.
Monitoring Sentry ✓ Free tier 5K errors/month free. Alerts on Telegram/email. Trace errors to exact line. Essential for production.
Analytics Vercel Analytics + PostHog ✓ Free tiers Core Web Vitals + product funnels + heatmaps. PostHog 1M events/month free.
Mobile App Flutter (your stack) ✓ Free Uses same REST API. BLoC + Clean Architecture — your existing expertise. Add in Phase 2.

Request Lifecycle

How Data Flows

Every key user journey — step by step from click to response.

01
Product Page Load
User visits /products/rose-oud-perfume
Cloudflare checks edge cache
HIT → HTML served in <50ms, done
MISS → Vercel runs ISR page
Check Upstash Redis for product
MISS → Supabase query via Drizzle
Store in Redis (TTL 10min)
Return HTML → Cloudflare caches
02
Product Search
User types "rose" in search bar
InstantSearch sends to Typesense
Typesense returns in <30ms
Filters: price, category, size
Zero DB hit — search is isolated
Supabase → Typesense sync on product update
03
Add to Cart + OTP Login
Anonymous checkout flow
Cart saved in Zustand + Redis
Checkout → enter phone number
POST /api/auth/otp/send → MSG91
User enters OTP → verify
Better Auth creates session (JWT)
Silent user row in Supabase
Anon cart merged to user cart
04
Payment Flow
Razorpay UPI / Card
POST /api/payments/create
Backend creates Razorpay order
Frontend opens Razorpay modal
User pays — Razorpay callback
POST /api/webhooks/razorpay
HMAC verify signature
Order status → PAID in Supabase
05
Post-Payment Notify
Multi-channel confirmation
Webhook triggers notify function
Resend → order email (React template)
MSG91 → SMS confirmation
WhatsApp API → order + image
Shiprocket API → create shipment
Tracking ID → stored in Supabase
06
Admin Dashboard
admin.flawlessera.store
RBAC check → admin role only
Product CRUD → Supabase + Typesense sync
Image upload → Cloudinary
Order management → status update
Analytics embed — PostHog
Invalidate Redis cache on update

Key Principle

What Never Changes

Architecture that survives all 4 phases without a rewrite.

Framework
Next.js 15 throughout
Same codebase from day 1 to 10K/day. API routes scale with Vercel auto-scaling. No migration ever.
Database Schema
PostgreSQL stays
Just upgrade Supabase plan. Same schema, same queries. Read replicas added transparently.
Cache Strategy
Redis pattern stays
Upstash just scales with usage. Same code. Same TTL logic. No refactor needed.
Search
Typesense API stays
Self-hosted → Typesense Cloud is just a URL change. Zero code change.
Payments
Razorpay stays forever
Razorpay handles any scale. No migration. Just enable more payment methods as needed.
Flutter App
Same REST API
Flutter calls the same Next.js API routes. Add mobile app in Phase 2 with zero backend changes.
